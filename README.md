# LinkedIn Clone

A full-featured LinkedIn clone built with **React 19** and **Vite**, replicating core professional networking functionality with a modern frontend stack.

---

## 🚀 Tech Stack

| Layer | Technology |
|---|---|
| UI Framework | React 19 |
| Build Tool | Vite 7 |
| State Management | Redux Toolkit + React-Redux |
| Routing | React Router DOM v7 |
| Styling | Tailwind CSS v4 |
| Component Library | Ant Design v6 |
| HTTP Client | Axios |

---

## ✨ Features

- Professional user profiles and feeds
- Post creation and interaction (like, comment)
- Navigation and routing between core LinkedIn-style pages
- Global state management via Redux Toolkit
- Responsive UI using Tailwind CSS and Ant Design components
- API integration with Axios

## 🛠️ Getting Started

### Prerequisites

- Node.js v18 or higher
- npm v9 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/Caseyvlogger/LinkedInClone.git

# Navigate into the project directory
cd LinkedInClone

# Install dependencies
npm install
```

### Running the App

```bash
# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Other Scripts

```bash
# Build for production
npm run build

# Preview the production build
npm run preview

# Run ESLint
npm run lint
```
